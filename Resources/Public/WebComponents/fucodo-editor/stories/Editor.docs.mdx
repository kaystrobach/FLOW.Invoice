## Description

The `fucodo-editor` web component provides a simple Markdown editing experience. It renders
an editor UI for the content inside its inner `<textarea>`. In the demo story, a Save button
reads the current Markdown via the component\'s JS API and displays it below the editor.

This documentation page is powered by the same stories used in the canvas/controls, so the
interactive example reflects the live configuration.

Tag name: `fucodo-editor`

## Usage

<Source
  language="html"
  code={`<fucodo-editor class="custom">
  <textarea>
# Hello World!
This is a simple markdown editor.
- foo
- bar
  </textarea>
</fucodo-editor>`}
/>
